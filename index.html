About GlideX

GlideX is an advanced autonomous warehouse management robot designed for efficient item transportation, AI-based object recognition, and SLAM-powered navigation using Mecanum wheels. It is built for high maneuverability and full automation in smart warehouses.

Key Features

Mecanum Wheel Movement

Precise omnidirectional movement using 4-wheel independent control.

YDLIDAR SLAM

Environment mapping and navigation using T-mini Plus LiDAR.

AI Object Detection

YOLOv8 + OpenCV integration for real-time object verification.

Actuation System

Automated actuator and conveyor system for product loading/unloading.

Web Dashboard

Remote control and automation triggers via a Flask-based interface.

Hardware & Software

  • Raspberry Pi 5 (8GB)
  • YDLIDAR T-mini Plus
  • Pi Camera Module 3 NoIR
  • L298N Motor Driver
  • 4S2P Molicel Li-ion Battery Pack
  • Python, OpenCV, YOLOv8, Flask
